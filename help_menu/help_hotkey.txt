F1 	- изменить ЗВАНИЕ И ИМЯ в удостоверении

	NumPad0     - представился + показал удостоверение
	Alt+NumPad0 - пока пусто

	NumPad1     - взял+вернул паспорт (с проверкой розыска через базу)
	Alt+NumPad1 - взял+вернул водительское удостоверение
	Crtl+NumP1  - взял+вернул ПТС (паспорт машины)

	NumPad2     - разбить окно вытащить человека из машины и "заломать руку"
	Alt+NumPad2 - сорвать маску с человека (ТОЛЬКО если в наручиках /unmask)

	NumPad3     - надеть наручники
	Alt+NumPad3 - /tome повел человека за собой
	Ctlr+NumPad3- снять наручники

	NumPad4     - опустить шлагбаум на посту
	Alt+Numpad4 - /check проверка ременя безопасности

	NumPad5     - посадить задержаного в машину
	Alt+NumPad5 - выкинуть из машины (/eject )

	NumPad6     - дать розыск (если человек в МАСКЕ - РОЗЫСК ДАВАТЬ НЕЛЬЗЯ!!)
	Alt+NumPad6 - выписать штраф (не забудь СНАЧАЛА ВЗЯТЬ паспорт /pass)

	NumPad7     - арестовать/посадить, передать человека в кпз (тюрьму)
	Alt+NumPad7 - выдача номеров (для мл.лейтенант и выше)

	NumPad8     - /time для видео

	NumPad9     - фоторобот (если человек НЕ В МАСКЕ{!})
	Alt+NumPad9 - воинское приветствие

	F2          - обыск человека
	Alt+F2      - после обыска изъять запрещ предметы: ОРУЖИЕ, НАРК
	Ctrl+F2     - изъял водительское (мл.лейт и выше)
	F3          - Мегафон для погони в машине
	Alt+F3      - /vehoff Заглушить двигатель преступнику. ЭМИ пушка

	\           - заготовка для ответа в /rb рацию
	NumPad*     - дать денег
	Shift+NPad* - список сотрудников /members
	Ctrl+NPad*  - поиск преступника через gps /setmark (ПРОСТО ТАК НЕЛЬЗЯ)
	Ctrl+Alt+F  - ремонт машины /fix
	Alt+h       - аптечка /healme
	Alt+g       - передать аптечку /givechest

	Ctrl+ЛКМ    - добить лежачего дубинкой по РП "до приезда скорой"
	Alt+1,2,3,4,5,6,7,8 - УК, КоАП, Устав, ФЗ о полиции. (пока держишь альт, окно не закроется)
	Alt+R       - ПЕРЕЗАПУСК АНК. ОСТАНОВКА АНК.
